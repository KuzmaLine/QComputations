# ПО для квантовых вычислений на C++

В данном репозитории лежит программная реализация на C++ для моделирования квантовых экспериментов.
Все реализации лежат в папке src.

Make реализует 3 библиотеки: - (Не готово)
1. Для одного ядра - single (Требуется Intel MKL)
2. Для нескольких MPI ядер - mpi (Требуется Intel MKL, OpenMPI или IntelMPI)
3. Для нескольких узлов - mpi_cluster (Требуется полный пакет Intel OneApi BaseToolkit (бесплатный) + Intel DPCPP++ + Intel MPI + Intel Python из-за IntelMPI)

Разница между 2 и 3 библиотеками в том, что во 2 не реализована эффективное параллельное перемножение матриц (Та же скорость, что и у blas)

Компиляция программ с библиотекой также нетривиальна из-за Python. Примеры их компиляций будут описаны в makefile в папке examples. (Не готово!)

# Описание структуры программы
(Всё лишнее из ПО будет вскоре будет удалено)

Всё ПО находится в namespace QComputations!

Все hpp файлы оснащены комментариями с описанием, что делает каждая функция. (Кроме тех, в котором описание кроется в самом названии оной)
Примеры программ лежат в директории examples. - (Не готово!)

Файл config.hpp - для настройки поведения программы по умолчанию (Неудачное решение, которое скоро заменю)

Файлы matrix - реализация класса матриц. Является основой всех остальных файлов. (В разработке CSR_Matrix)

Файлы big_uint - реализация больших чисел. Используется для файлов state

Файлы cavity_state - состояние внутри одной полости. (То есть есть некоторое пространство, внутри которого находятся фотоны и атомы)

Файлы state - состояние решётки полостей. Подробности смотреть в файле state.hpp

Файлы hamiltonian - Сами гамильтонианы. Есть 6 реализаций:

1. H_JC - Гамильтониан Джеймса-Каммингса (1 полость, n фотонов, 1 атом)
2. H_TC - Гамильтониан Тависа-Каммингса (1 полость, n фотонов, m атомов)
3. H_TCH - Гамильтониан Тависа-Каммингса-Хабарда (k полостей, n фотонов, m атомов)
4. H_by_matrix - Гамильтониан, заданный с помощью комплексной матрицы.
5. H_by_func - Гамильтониан, заданного размера, реализованный с помощью функций.
6. H_Blocked - Гамильтониан, распределённый на блоки по разным процессам. (Только в mpi_cluster) (Не готово!)

Все гамильтонианы, кроме 4, поддерживают параллельную генерацию. Для 5 и 6 - отдельная 
инструкция в examples H_by_func_mpi_example.cpp и H_Blocked_MPI_example.cpp соответственно. (Не готово!)

Файлы dynamic - функции процесса эволюции. (Реализованы: Шрёдингер для статичного гамильтониана, Эволюция с помощью основного квантового уравнения)
Для обоих есть параллельные реализации. (ВАЖНО! - для параллельного основого квантово уравнения нужен mpi_cluster и прочесть dynamic.hpp)

Файлы functions - дополнительные полезные функции. Подробнее в functions.hpp

Файлы graph - поиск дерева состояний для составления базиса (В разработке для решётки)

Файлы plot - для генерации графиков (На стадии улучшения, на данный момент реализован matplotlib)

Файлы quantum_operators - функции эволюции заданного состояния под воздействием некоторых стандартных гамильтонианов. Подробнее в quantum_operators.hpp
